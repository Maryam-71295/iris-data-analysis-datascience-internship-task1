# iris-data-analysis-datascience-internship-task1
The dataset is sourced from Kaggle and contains 150 observations of iris flowers across three species: Setosa, Versicolor, and Virginica. Each observation includes measurements such as sepal length, sepal width, petal length, and petal width. 
📌 Objective
The objective of this project is to explore and visualize the Iris dataset and understand relationships between different flower features.

📊 Dataset
- Source: Kaggle (https://www.kaggle.com/datasets/uciml/iris/data)
- Records: 150
- Features: 4 numerical + 1 categorical

⚙️ Approach
- Data loading using pandas
- Data cleaning (removed unnecessary columns)
- Exploratory Data Analysis (EDA)
- Visualization using seaborn and matplotlib

📈 Key Insights
- Petal length and width are strong predictors
- Setosa species is easily separable
- Dataset is clean with no missing values

🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
